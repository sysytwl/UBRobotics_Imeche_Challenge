2. The following pins are internally connected to the microSD card reader:
  - GPIO 14: CLK
  - GPIO 15: CMD
  - GPIO 2: Data 0
  - GPIO 4: Data 1 (also connected to the on-board LED)
  - GPIO 12: Data 2
  - GPIO 13: Data 3
